# Component Props
You can also use any native react props or html props as desired (for example `onClick`).
The button component comes with a polymorphic `as` prop, to enable you to render to any html element you need (e.g. `as="a"`).
Ref is being forwarded to the `HTMLElement`.

<Props of={Button} />

# Variants

Never use more than ONE primary button on a page-view.
The primary button should help the user in his actions by leading him on a page-view.
Use carefully a primary button, only when it is needed and do not mislead the user to unwanted actions.
